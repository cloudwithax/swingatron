# Swingatron

![](https://raw.githubusercontent.com/cloudwithax/swingatron/refs/heads/master/resources/preview.png)

[![License](https://img.shields.io/badge/license-BSD_3_Clause-orange?style=flat-square)](https://github.com/cloudwithax/swingatron/blob/master/LICENSE) [![GitHub tag](https://img.shields.io/github/v/release/cloudwithax/swingatron?include_prereleases=&sort=semver&color=blue&style=flat-square)](https://github.com/cloudwithax/swingatron/releases) [![issues - swingatron](https://img.shields.io/github/issues/cloudwithax/swingatron?style=flat-square)](https://github.com/cloudwithax/swingatron/issues)

Swingatron is the most comprehensive music player for Swing Music, built with Electron and Vue.

## Features

- Browse and play music from your Swing Music server
- Create and manage playlists
- Favorite tracks and albums
- Search your music library
- Scan library for new music additions
- View detailed information and metadata about tracks and albums
- See synced lyrics for most tracks
- Last.fm scrobbling support
- Fullscreen mode with cover flow view
- Native media controls and integration with your operating system
- Discord Rich Presence support

## Installation

Download the latest release for your operating system from the [Releases](https://github.com/cloudwithax/swingatron/releases) page.
